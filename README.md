# Shopping List Manager

## Overview

This project is a Shopping List Manager application created to practice JavaScript programming concepts. The application allows users to add items to a shopping list, remove items, mark items as completed, and clear the entire list.

The purpose of this software is to improve my understanding of JavaScript fundamentals, including variables, arrays, objects, functions, events, and DOM manipulation. This project demonstrates how JavaScript can be used to create interactive web applications.

## Features

- Add new shopping items
- Remove items from the list
- Mark items as completed
- Clear all shopping items
- Display the total number of items
- Add items using the Enter key
- Prevent duplicate items
- Show the date each item was added (month/day format, example: 7/12)
- Show a message when the list is empty

## How It Works

1. Type an item name and click Add Item (or press Enter).
2. The app checks for duplicates before adding.
3. A new item is saved with its completion state and added date.
4. Complete and Remove buttons let the user manage each item.
5. The item counter updates after every change.
6. Clear All asks for confirmation before deleting everything.
